# Projeto Agrinho 2026 – Terra Indígena Queimadas

Este projeto é um site escolar desenvolvido para o Projeto Agrinho 2026.  
O tema escolhido é a comunidade onde vivem os estudantes: a Terra Indígena Queimadas, localizada no município de Ortigueira, no Paraná.

## Objetivo do projeto

Apresentar, de forma respeitosa e educativa, informações sobre a comunidade, sua história, seu território, seus modos de vida, seus saberes, seus desafios e sua relação com a natureza.

## Arquivos do projeto

- `index.html`: estrutura principal do site.
- `style.css`: estilos visuais do site.
- `script.js`: interações simples com JavaScript.
- `img/`: pasta usada para guardar as imagens do site.
- `img/araucaria.jpg`: imagem de fundo da página inicial.
- `README.md`: explicação do projeto.

## Estrutura de pastas

```text
projeto-agrinho-2026/
├── README.md
├── index.html
├── style.css
├── script.js
└── img/
    └── araucaria.jpg
```

## Como visualizar o site

1. Baixe ou copie os arquivos do projeto.
2. Abra o arquivo `index.html` no navegador.
3. Para publicar no GitHub Pages:
   - Crie um repositório no GitHub.
   - Envie os arquivos do projeto.
   - Vá em **Settings > Pages**.
   - Em **Branch**, selecione `main` e a pasta `/root`.
   - Clique em **Save**.

## Como trocar a imagem de fundo

A imagem de araucária usada como fundo da página inicial está salva na pasta `img/` com o nome `araucaria.jpg`.

Para trocar a imagem:

1. Salve a nova foto dentro da pasta `img/`.
2. Dê um nome simples para a imagem, por exemplo: `comunidade.jpg`.
3. Abra o arquivo `style.css`.
4. Troque esta linha:

```css
url("img/araucaria.jpg");
```

por:

```css
url("img/comunidade.jpg");
```

## Orientação importante

Antes de publicar fotos, nomes de pessoas, histórias, rituais, cantos, conhecimentos tradicionais ou informações internas da comunidade, converse com os professores, familiares, lideranças e responsáveis.  
O site deve valorizar a comunidade sem expor conteúdos sensíveis ou que não devem ser divulgados publicamente.
