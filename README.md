# Projeto Agrinho 2026 – Terra Indígena Queimadas

Este projeto é um site escolar desenvolvido para o Projeto Agrinho 2026.  
O tema escolhido é a comunidade onde vivem os estudantes: a Terra Indígena Queimadas, localizada no município de Ortigueira, no Paraná.

## Objetivo do projeto

Apresentar, de forma respeitosa e educativa, informações sobre a comunidade, sua história, seu território, seus modos de vida, seus saberes, seus desafios e sua relação com a natureza.

## Arquivos do projeto

- `index.html`: estrutura principal do site.
- `style.css`: estilos visuais do site.
- `script.js`: interações simples com JavaScript.
- `img/`: pasta usada para guardar as imagens do site.
- `README.md`: explicação do projeto.
